## 🎨 Badges Personnalisés

Vous pouvez personnaliser les badges en modifiant l'URL :

### Métriques Disponibles

- `alert_status` - Quality Gate Status
- `coverage` - Code Coverage
- `bugs` - Number of Bugs
- `vulnerabilities` - Number of Vulnerabilities
- `code_smells` - Number of Code Smells
- `security_rating` - Security Rating (A-E)
- `sqale_rating` - Maintainability Rating (A-E)
- `reliability_rating` - Reliability Rating (A-E)
- `sqale_index` - Technical Debt
- `duplicated_lines_density` - Duplicated Lines (%)
- `ncloc` - Lines of Code

### Format de l'URL

```
https://sonarcloud.io/api/project_badges/measure?project={PROJECT_KEY}&metric={METRIC}
```

Remplacez :
- `{PROJECT_KEY}` par votre clé de projet (ex: `user-microservice`)
- `{METRIC}` par la métrique souhaitée
