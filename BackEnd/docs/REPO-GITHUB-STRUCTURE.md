# Structure du repo GitHub – PIDEV-4eme

Repo : **https://github.com/khadijaba/PIDEV-4eme.git**

## Structure cible (branche master)

```
PIDEV-4eme/
├── BackEnd/
│   ├── Eureka/
│   │   └── eureka/              # Serveur Eureka
│   ├── ApiGateway/              # API Gateway
│   ├── Microservices/
│   │   ├── Skill/
│   │   ├── Project/
│   │   └── User/                # (à ajouter par le membre User)
│   └── docs/                    # ARCHITECTURE-EQUIPE, GIT-WORKFLOW, etc.
├── Frontend/                    # Application Angular (frontend)
│   ├── src/
│   ├── public/
│   ├── angular.json
│   ├── package.json
│   └── ...
└── documentation/               # (existant dans le repo)
```

## Chemins locaux à utiliser

| Dans le repo (GitHub) | Sur ta machine (exemple) |
|------------------------|--------------------------|
| `BackEnd/`             | Contenu de `BackEnd\BackEnd\` (Eureka, ApiGateway, Microservices, docs) |
| `Frontend/`            | Contenu de ton dossier Angular (ex. `frontendd\frontend`) |
