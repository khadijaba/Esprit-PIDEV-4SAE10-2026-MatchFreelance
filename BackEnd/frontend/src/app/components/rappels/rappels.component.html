Rappels

Vue des rappels générés par le script Python (formations ouvertes, examens à passer, formation se termine, certificats).

Mise à jour manuelle : dans le dossier python, python rappels.py --export (affichage) ou python rappels.py --auto (export + envoi des emails).

Rappel automatique

Pour envoyer les rappels (et mettre à jour cette page) automatiquement chaque jour :

  • 1. Configurez SMTP dans python/.env (SMTP_HOST, SMTP_USER, SMTP_PASSWORD, SMTP_FROM).
  • 2. Windows : Planificateur de tâches → Créer une tâche → Déclencheur quotidien (ex. 9h) → Action : exécuter run_rappels_auto.bat dans le dossier python.
  • 3. Linux / Mac : crontab 0 9 * * * cd /chemin/vers/python && python rappels.py --auto (tous les jours à 9h).

La commande python rappels.py --auto met à jour le fichier des rappels (cette page) puis envoie les emails aux freelancers. La Gateway et les microservices doivent être accessibles au moment de l’exécution.
